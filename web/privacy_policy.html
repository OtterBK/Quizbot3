‘퀴즈봇3’은 디스코드 개인 정보 보호 방침에 따라 사용자의 개인정보와 그 권리를 보호하고자
다음과 같은 처리방침을 두고 있습니다.


1. 개인정보의 처리 목적
     ‘퀴즈봇3’은 개인정보를 다음의 목적을 위해 처리합니다.
    처리한 개인정보는 아래 목적 이외의 용도로는 사용되지 않습니다.
    만일 이용 목적이 변경될 시에는 사전동의를 구할 예정입니다.

   ㆍ제공 서비스: [순위표, 사용자 퀴즈 관리, 정답 여부 확인] 기능에서 개인정보를 처리합니다.


2. 개인정보 수집현황

    수집하는 개인정보는 아래와 같습니다.

  
  ㆍ항목 :
     – (필수항목) Discord 사용자 ID | 외부 저장매체(데이터베이스)에 저장됨
     – (선택항목) 입력한 채팅 메시지 | 외부 저장매체에 저장되지 않고 프로세스 메모리 상에서만 사용됨

    ㆍ수집방법 :
    퀴즈봇3로 퀴즈 플레이를 완료
    퀴즈봇3에서 사용자 퀴즈 생성
    퀴즈 진행 중인 채널에서 채팅 메시지 입력

    * 옵션에서 [메시지 입력 감지] 옵션을 비활성화하여 채팅 메시지 정보 수집을 거부할 수 있습니다.



3. 개인정보의 처리 및 보유 기간
    Discord 사용자 ID | 사용자의 요청이 있을 경우 즉시 파기됩니다.
    (이 경우, 해당 사용자가 만든 퀴즈 및 해당 사용자의 순위 데이터도 전부 소멸되며 복구가 불가능합니다.)


    채팅 메시지 | 퀴즈가 진행 중인 채팅 채널에서만 메시지를 수집하며 입력된 메시지는
    퀴즈의 정답 여부만 확인한 뒤, 프로세스 메모리 상에서 소멸됩니다.


4. 개인정보의 제3자 제공에 관한 사항
    ‘퀴즈봇3'은 제3자에게 정보 제공을 하지 않습니다.


5. 개인정보의 파기
    개인정보 처리목적이 달성된 경우나 정보주체가 요구하는 경우에는
    지체없이 해당 개인정보를 파기하며 그 방법은 다음과 같습니다.

    ㆍ파기방법: 데이터베이스에서 사용자의 개인정보 영구 삭제


6. 개인정보 처리방침 변경 고지
    변경내용의 추가, 삭제 및 정정이 있는 경우에는 변경사항의 시행 7일 전부터 공지사항을 통하여 고지할 것입니다.
